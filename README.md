# Sales-Analysis-Dashboard

📊 Project Overview

This project presents a Sales Analysis Dashboard built using Power BI to analyze business performance.

🔧 Tools Used

- Power BI
- DAX
- Power Query

📈 Key Insights

- Sales performance by city and country
- Quarterly trends
- Top-performing locations

📌 Features

- KPI Cards (Total Sales, Quantity, Profit)
- Interactive filters (Quarter, City)
🚀 How to Use

Download the .pbix file and open it in Power BI Desktop.
